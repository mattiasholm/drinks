# Colorado Bulldog

## Ingredients
- 3 cl `vodka`
- 3 cl `coffee liqueur`
- 3 cl `cream`
- 3 cl `Coca-Cola`
- 16 `ice cubes`

## Instructions
- Add vodka
- Add coffee liqueur
- Add cream
- Add ice cubes and shake for 15 seconds
- Double strain
- Add ice cubes
- Add Coca-Cola

## Glassware
- Old Fashioned glass
