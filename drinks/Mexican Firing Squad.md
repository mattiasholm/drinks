# Mexican Firing Squad

## Ingredients
- 5 cl `tequila blanco`
- 3 cl `lime juice`
- 2 cl `grenadine`
- 2 dashes `Angostura Aromatic Bitters`
- 16 `ice cubes`

## Instructions
- Add Angostura Aromatic Bitters
- Add tequila blanco
- Add lime juice
- Add grenadine
- Add ice cubes and shake for 15 seconds
- Double strain
- Add ice cubes

## Glassware
- Old Fashioned glass
