# Only The Good Die Young

## Ingredients
- 3 cl `dark rum`
- 2 cl `elderflower liqueur`
- 3 cl `lime juice`
- 2 cl `maple syrup`
- 16 `ice cubes`

## Instructions
- Add dark rum
- Add elderflower liqueur
- Add lime juice
- Add maple syrup
- Add ice cubes and shake for 15 seconds
- Double strain
- Add ice cubes

## Glassware
- Old Fashioned glass

## Rating
- ★★★★★
