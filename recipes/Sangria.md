# Sangria

## Ingredients:
- 75 cl `red wine`
- 50 cl `7 Up`
- 25 cl `orange juice`
- 2 `oranges`
- 2 `apples`
- 32 `ice cubes` <!-- - 8 `ice cubes` -->

## Instructions:
- Add red wine
- Add 7 Up
- Add orange juice
