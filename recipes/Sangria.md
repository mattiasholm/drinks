# Sangria

## Ingredients:
- 75 cl `red wine`
- 10 cl `vodka`
- 50 cl `7 Up`
- 20 cl `orange juice`
- 2 `oranges`
- 2 `apples`
- 32 `ice cubes`

## Instructions:
- Add red wine
- Add vodka
- Add 7 Up
- Add orange juice
- Add sliced oranges
- Add sliced apples
- Add ice cubes and stir

## Served in:
- Pitcher

## Rating:
- ★★★★☆
